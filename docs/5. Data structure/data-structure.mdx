# 📦 Data Structures

Store multiple values together

---

## 🧠 Beyond Single Values

So far, you’ve stored one value per variable.

👉 But what if you need **multiple values together?**

That’s where **data structures** come in.

Think of them like containers:

- **Lists** → shopping list (ordered items)  
- **Dictionaries** → phone book (name → number)  
- **Tuples** → coordinates (fixed values)  
- **Sets** → unique items only

## 🧭 Which One to Use?

- **Lists** → order matters + editable  
- **Dictionaries** → fast lookup by key  
- **Tuples** → fixed data  
- **Sets** → unique values  

---

💡 Python uses **zero-based indexing**  
(first item = position 0)
