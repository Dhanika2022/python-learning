# 🔀 Control Flow

Make your programs smart with decisions

---


## 🧠 Making Programs Think

So far, your programs run **top to bottom**, executing every line.

👉 But real programs need to make decisions:

- “If this happens…”  
- “Then do something else…”  

👉 This is called **control flow**

---

## ⚡ What is Control Flow?

Control flow is what makes programs **intelligent**.

It allows your code to:
- Make decisions  
- Respond to different situations  
- Change behavior dynamically  

---

## 🎯 What You’ll Learn

### 🔀 If Statements  
Make decisions based on conditions

## 🌍 Real-World Examples

Decision-making is everywhere:

- 🏧 ATM → IF password is correct → allow access  
- 📱 Apps → IF button clicked → perform action  
- 🌡 Weather app → IF temperature < 0 → show snow  
- 🎮 Games → IF health = 0 → game over
