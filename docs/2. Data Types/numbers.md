# 🔢 Numbers

Working with integers and decimals

---

## 🧠 Two Types of Numbers

Python has two main types of numbers:

---

### 🔢 Integers (int)

Whole numbers without decimals:

```python
age = 25
score = -10
```

---

### 🌊 Floats (float)

Numbers with decimal points:

```python
price = 19.99
temperature = -5.5
pi = 3.14159
```

⚠️ Python uses a **dot (.)**, not a comma  
```python
3.14   # ✅ correct
3,14   # ❌ creates a tuple
```

---

## ⚡ Why Two Types?

Python separates integers and floats for efficiency.

👉 But don’t worry too much —  
Python automatically handles conversions when needed.

✔ Use integers → for counting  
✔ Use floats → for measurements  

---

## 🧮 Basic Math Operations

```python
# Addition and subtraction
total = 10 + 5     # 15
change = 20 - 7    # 13

# Multiplication and division
area = 6 * 4       # 24
half = 10 / 2      # 5.0

# Powers
squared = 5 ** 2   # 25
cubed = 2 ** 3     # 8
```

---

## 🔍 Integer vs Float Division

```python
# Regular division → always float
result = 10 / 3    # 3.333...

# Integer division → rounds down
result = 10 // 3   # 3
```

---

## ⚠️ Common Mistakes

- Division always returns float  
- Using commas in numbers (`3,14`)  
- Confusing `/` and `//`  

---

## 🧾 Quick Summary

- Integers → whole numbers  
- Floats → decimal numbers  
- `/` → float division  
- `//` → integer division
