# ⚙️ Functions

Create reusable blocks of code

---


## 🧠 Building with Functions

Functions are reusable blocks of code that perform a specific task.

👉 Instead of writing the same code multiple times,  
👉 you write it once and reuse it whenever needed  

---

### 📌 Think of functions like:

- 📖 A recipe you can reuse  
- ⚙️ A machine (input → output)  
- 🔁 A shortcut for repeated logic  

---

## 🎯 Why Use Functions?

- ✅ **Avoid repetition** → write once, reuse  
- 🧩 **Stay organized** → break code into small parts  
- 🐞 **Fix bugs easily** → update in one place  
- 🧪 **Test easily** → test each function independently
